SETTING THE KEYBOARD LAYOUT
________________________________________________________________________________


Acquiring keymaps from the bkeymaps package
________________________________________________________________________________

You can get the bkeymaps package from Community.

+------------------------------------------------------------------------------+
|                                                                              |
|   $ kiss b bkeymaps                                                          |
|   $ kiss i bkeymaps                                                          |
|                                                                              |
+------------------------------------------------------------------------------+

Now you can find the keymaps in /usr/share/bkeymaps.


Loading Keymaps at login
________________________________________________________________________________

To load a keymap at login, you can add the command above to your .profile file.

+------------------------------------------------------------------------------+
|                                                                              |
|   $ loadkmap < file                                                          |
|                                                                              |
+------------------------------------------------------------------------------+
